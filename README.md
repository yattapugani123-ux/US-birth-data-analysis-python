US Birth Data Analysis using Python (Pandas, Matplotlib, Seaborn)
# US Birth Data Analysis (1994–2003)

This project analyzes US birth data using Python.

## Questions Answered
- What is the total number of births?
- What is the average number of births per day?
- Which month has the highest births?
- Which day of the week has the most births?

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Visualizations
- Births by month (Bar chart)
- Births by weekday (Bar chart)
- Birth trend by year (Line chart)
- Birth heatmap (Month vs Weekday)
